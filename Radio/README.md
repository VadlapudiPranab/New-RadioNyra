# Radionayra

This is a clone of the Radio Nyra website with a clean white-based theme.
Created as part of a redesign project.

## Features
- **White Theme**: Clean, modern UI with `#E91E63` accent.
- **Responsive Design**: Mobile-friendly layout.
- **Grid Layout**: Gallery converted to CSS Grid.
- **Pages**: Home, Shows, Events, Gallery, Contact.

## Setup
To run locally:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`
